# Ishimwe Xavion - Personal Portfolio Website

Welcome to my personal portfolio website! This is a modern, responsive website built with HTML, CSS, and JavaScript.

## Features

- **Navigation Bar**: Easy navigation to all sections
- **Hero Section**: Eye-catching introduction
- **CV Section**: Display your education, experience, skills, and certifications
- **Links Section**: Direct link to your Linktree and social media
- **Calendar Widget**: Interactive calendar showing the current month
- **Weather Widget**: Real-time weather information based on your location
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## Contents

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `script.js` - Calendar generation and weather functionality
- `README.md` - Documentation

## How to Use

1. Clone this repository
2. Customize the content in `index.html`:
   - Update your CV details
   - Add your email address
   - Update any other personal information
3. Open `index.html` in your browser to view your website
4. Deploy to GitHub Pages or any hosting service

## Customization

### Update CV Section
Edit the CV items in the `<section id="cv">` to reflect your actual education, experience, skills, and certifications.

### Add Your Contact Information
- Update the email link in the Links section
- Add more social media links as needed

### Weather Widget
The weather widget uses the free Open-Meteo API and requests your browser location for accurate local weather. If you deny location access, it defaults to Rwanda coordinates.

### Calendar Widget
The calendar automatically generates based on the current month and highlights today's date.

## Deployment

To deploy on GitHub Pages:

1. Ensure your repository is named `Etchebbeari.github.io`
2. Push the files to the main branch
3. Access your website at `https://Etchebbeari.github.io`
